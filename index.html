AI WYRM API Documentation

Endpoint

AI WYRM API, platformu üzerinden wyrm_1 modelini kullanır.

https://wyrm.com/console/v1/chat/completions

Örnek Kullanım

Aşağıdaki cURL komutu ile API'yi test edebilirsiniz:

curl https://wyrm.com/console/v1/chat/completions -s \ -H "Content-Type: application/json" \ -H "Authorization: Bearer api gizli partnerlik alın" \ -d '{ "model": "WYRM/gdb3.1/wyrm.1", "messages": [{ "role": "user", "content": "Explain the importance of fast language models" }] }'

Model Bilgileri

• Model: wyrm.1
• Hızlı yanıt süresi
• Türkçe dil desteği
• SquaerdsSOFT tarafından özelleştirilmiş

Parametreler

{ "model": "wyrm.b543nfr.54fb_1vrs", "messages": [ { "role": "system", "content": "Sen AI WYRM'sin..." }, { "role": "user", "content": "Kullanıcı mesajı" } ], "temperature": 0.7, "stream": false }
