AI WYRM - Yapay Zeka API

AI API Endpoint

AI WYRM, gelişmiş yapay zeka teknolojisi ile güçlendirilmiş API hizmeti sunar.

https://api.wyrm.com/console/v1/chat/completions

Yapay Zeka API Kullanımı

AI WYRM API'sini kullanarak yapay zeka destekli uygulamalar geliştirebilirsiniz:

curl https://api.wyrm.com/console/v1/chat/completions -s \ -H "Content-Type: application/json" \ -H "Authorization: Bearer GİZLİ" \ -d '{ "model": "wyrm/wyrm-1-scout-17b-16e-instruct", "messages": [{ "role": "user", "content": "Explain the importance of fast language models" }] }'
